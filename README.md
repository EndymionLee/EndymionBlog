# 🚀 Endymion's Blog

李杨明的个人技术博客，记录 AI Agent、Go 后端、Flutter 跨平台开发的学习与思考。

## 技术栈

- **框架**: [VitePress](https://vitepress.dev)
- **部署**: GitHub Pages + GitHub Actions
- **域名**: yourname.dev

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态页面
npm run build

# 预览构建结果
npm run preview
```

## 目录结构

```
docs/
├── AI/          # AI Agent 相关
├── Go/          # Go 后端
├── Flutter/     # Flutter 跨平台
├── Python/      # Python
├── Architecture/# 架构设计
├── Project/     # 项目日志
├── Weekly/      # 周记
├── public/      # 静态资源
├── about.md     # 关于我
├── projects.md  # 项目展示
└── resume.md    # 简历
```

## 部署

push 到 main 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。
